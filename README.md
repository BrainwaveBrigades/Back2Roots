### 🚀 How to Run Locally
1. Clone this repository:
   ```sh
   git clone https://github.com/BrainwaveBrigades/Back2Roots.git
   cd Back2Roots
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```
